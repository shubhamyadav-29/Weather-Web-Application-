# 🌦️ Weather Web Application

A modern and responsive **Weather App** built using **React + Vite** that allows users to search real-time weather data, view forecasts, and use their current location.

---

## 🚀 Live Demo

🔗 https://weather-web-application-psi.vercel.app/

---

## 📂 GitHub Repository

🔗 https://github.com/shubhamyadav-29/Weather-Web-Application-

---

## ✨ Features

* 🔍 Search weather by city name
* 📍 Get weather using current location (Geolocation API)
* 🌡️ Real-time temperature, humidity, wind speed
* 📅 5-day weather forecast
* ❤️ Add/remove favorite cities
* 💾 Favorites stored using LocalStorage
* 📱 Fully responsive (Mobile + Desktop)
* ⚡ Fast performance using Vite

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite
* **API:** OpenWeatherMap API
* **Styling:** CSS (Glassmorphism UI)

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/shubhamyadav-29/Weather-Web-Application-

# Navigate to project folder
cd Weather-Web-Application-/weather-app

# Install dependencies
npm install

# Run the development server
npm run dev
```

---

## 🔑 API Configuration

1. Get your API key from: https://openweathermap.org/api
2. Replace API key in:

```js
src/services/weatherService.js
```
